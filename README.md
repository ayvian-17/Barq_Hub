# BarqHub4U ⚡

**BarqHub4U** is a premium, modern, and highly interactive landing page for professional electrical services. Designed with a dark mode aesthetic, it features advanced micro-interactions, smooth scrolling animations, dynamic WhatsApp messaging integration, and custom animations.

---

## 🚀 Live Demo

You can view the live site here: **[ayvian-17/BarqHub.github.io](https://ayvian-17.github.io/Barq_Hub.github.io/)** *(Note: Update with your exact GitHub Pages live URL)*

---

## ✨ Features

- **Dynamic Particle Background**: Uses `particles.js` to render interactive glowing ambient nodes that respond to hover and click inputs.
- **Custom Cursor & Follower**: Includes a custom glowing cursor with scale-up hover animations when interacting with buttons, links, and forms.
- **Interactive Review System**: Users can rate services using an interactive 5-star input and submit custom feedback.
- **WhatsApp Integration**:
  - A persistent **Floating WhatsApp Widget** with a hover tooltip.
  - A dynamic **Service Booking Form** that parses user inputs (Name, Phone, Service Type) and automatically redirects them to WhatsApp with a pre-filled message.
- **Interactive Stats Counter**: Animated numbers highlighting business stats (Happy Clients, Support hours, Years of Experience) that count up when scrolled into view.
- **Sliding Testimonial Carousel**: Touch-friendly, automatic, and manual sliding carousel for customer reviews with pagination indicator dots.
- **Smooth Animations**: Sleek scroll-triggered entry transitions, custom navbar scrolling transitions, and preloader animations.

## 🛠️ Technologies Used

- **HTML5**: Structured semantic markup.
- **CSS3 (Vanilla)**: Fully customized premium design layout utilizing CSS variables, transitions, and flexbox/grid.
- **JavaScript (ES6+)**: Custom animations, modal reviews, WhatsApp API redirection, and slide controls.
- **Particles.js**: Custom background canvas.
- **FontAwesome**: Scalable vector icons.
- **Google Fonts**: 'Poppins' typography family.

## 🛠️ How to Run Locally

1. **Clone the Repository**:
   ```bash
   git clone https://github.com/ayvian-17/BarqHub.github.io.git
   ```
2. **Navigate to the Directory**:
   ```bash
   cd BarqHub.github.io
   ```
3. **Open the project**:
   - Double-click `index.html` to open it in your browser.
   - Alternatively, serve it locally using VS Code's **Live Server** extension.

## 🌍 Deployment (GitHub Pages)

This project is configured to run out of the box using GitHub Pages:

1. Push this codebase to your GitHub repository named `ayvian.github.io` (or a sub-folder).
2. Go to **Settings** > **Pages** in your GitHub repository settings.
3. Under **Build and deployment**, set the source to `Deploy from a branch` and select `master` (or `main`) as the branch, folder `/ (root)`.
4. Save the configuration and wait for the deployment build to complete.
